# Mangalam Pipes - Industrial Landing Page

A premium, responsive landing page for **Mangalam Pipes**, a leading manufacturer of HDPE Pipes and Fittings. This project showcases a modern industrial web interface with dynamic components, technical data presentation, and a clean user experience.

## ✨ Features

### 🛠 Technical Highlights
* **Dynamic Gallery:** Interactive product image slider with thumbnail navigation and full-screen preview logic.
* **Sticky Navigation:** Smart navbar that hides on scroll-down and reveals on scroll-up to maximize screen real estate.
* **Custom Accordion:** Smooth-transition FAQ section for improved user engagement.
* **Infinite Testimonial Slider:** Auto-scrolling client feedback loop designed for social proof.
* **Process Tabs:** A step-by-step interactive breakdown of the HDPE manufacturing process.

### 📱 User Experience
* **Fully Responsive:** Optimized for Desktop, Tablet, and Mobile using CSS Grid and Flexbox.
* **Industrial Aesthetics:** Clean typography (Inter/Urbanist) paired with a professional blue-centric color palette.
* **Conversion Focused:** strategically placed "Get Custom Quote" and "Request Catalogue" CTAs.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Custom properties (variables), Grid, Flexbox, and Keyframe animations.
* **JavaScript (Vanilla):** DOM manipulation for the gallery, carousel, and tabbed interfaces.
* **FontAwesome:** High-quality industrial iconography.
